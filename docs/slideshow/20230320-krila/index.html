CLI → GUI → CLI (Prompt)

Microsoft 365 Copilot은 Apps와 그래프와 LLM이 유기적으로 동작하여 최선의 결과를 제시함.

미팅 노트, 작성 문서, 템플릿 등을 넣어주면 워드문서를 자동생성하고 “Summary” 도 자동으로 생성하여 삽입.

엑셀에 나와 있는 데이터를 다양한 표와 그래프로 생성하고 분석결과에 대한 인사이트도 제공

제안서 문서가 있는 경우 이를 PPT로 초안을 자동 생성함.

전자우편에 담긴 정보를 분석하여 제공하고 초안도 작성함.

팀즈를 통한 회의가 진해오디면 회의록 작성, Action Item, 업무 담당자 지정 등을 부조종사가 대략적인 업무를 정리함.

관련 정보를 모두 가져와서 사용자가 제시하는 질문에 대답을 하고 방향도 제시함.

미팅에 늦게 들어갔을 경우 채팅창에 부조종사를 호출하여 진행된 미팅에 대한 전반적인 사항을 파악할 수 있음.

구글 A new era for AI and Google Workspace

Google Worksapce에 AI (GPT)를 탑재하는 홍보영상을 게시하였으나 실제 사용까지는 다소 시간이 걸릴 것으로 보인다.

  • 적용제품
  • 업무방식
  • 구글 Docs
  • 구글 Gmail
  • Gmail: 초안 작성, 답장, 요약 및 우선순위 지정
  • Docs: 브레인스토밍, 교정, 작성, 재작성
  • Slide: 자동 생성된 이미지, 오디오, 동영상으로 창의적인 작업
  • Sheets: 자동 완성, 수식 생성, 상황별 분류를 통해 원시 데이터에서 인사이트와 분석.
  • Meet: 새로운 배경을 생성하고 메모를 캡처
  • Chat: 작업을 완료하기 위한 워크플로우 활성화
