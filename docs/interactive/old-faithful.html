eruption_tbl = FileAttachment("../data/faithful.csv").csv({ typed: true })

데이터 패키지

R 혹은 파이썬에서 데이터를 바로 생성시키고 이를 인터랙티브 시각화에 사용하는 방식도 존재한다. 마찬가지로 외부 데이터 패키지 bitData 에서 R 환경에서 데이터를 적재시키고 이를 인터랙티브 시각화에 사용하는 방식도 있다.

library(tidyverse)

normal_values <- rnorm(1000)

normal_data <- normal_values %>% 
  as_tibble()

ojs_define(normal_data = normal_data)

# 데이터

데이터는 외부에서 `.csv` 파일등을 가져오는 방식이 대표적이다. 또다른 방식은 데이터 패키지 ([`bitData`](https://github.com/bit2r/bitData))에 내장된 데이터를 바로 가져오는 방식이다.

## 외부 데이터

`faithful.csv` 로컬 디스크에 저장된 데이터를 가져와서 인터랙티브 시각화에 사용하는 경우 다음과 같이 `FileAttachment()` 함수를 사용한다.

```{ojs}
#| eval: false
eruption_tbl = FileAttachment("../data/faithful.csv").csv({ typed: true })

<!-- ojs_define(eruption_data = eruption_tbl) -->
```

## 데이터 패키지

R 혹은 파이썬에서 데이터를 바로 생성시키고 이를 인터랙티브 시각화에 사용하는 방식도 존재한다. 마찬가지로 외부 데이터 패키지 [`bitData`](https://github.com/bit2r/bitData) 에서 R 환경에서 데이터를 적재시키고 이를 인터랙티브 시각화에 사용하는 방식도 있다.


```{r}
#| eval: false
#| output: false
#| warning: false
#| message: false
#| echo: false

library(tidyverse)

normal_values <- rnorm(1000)

normal_data <- normal_values %>% 
  as_tibble()

ojs_define(normal_data = normal_data)
```

# UI

인터랙티브 데이터 시각화를 위해서 먼저 UI 입력부분을 정의힌다.

```{ojs}
//| panel: input
viewof bill_length_min = Inputs.range(
  [32, 50], 
  {value: 35, step: 1, label: "Bill length (min):"}
)
viewof islands = Inputs.checkbox(
  ["Torgersen", "Biscoe", "Dream"], 
  { value: ["Torgersen", "Biscoe"], 
    label: "Islands:"
  }
)
```
